# 📋 Advanced Todo Reminder App

A responsive Todo List application built with React that allows users to:

- Add tasks with date and time
- Receive browser notifications when task time arrives
- Edit tasks
- Mark tasks as completed
- Delete tasks
- Persist data using localStorage
- Responsive mobile-friendly design

## 📦 Features

### ✅ Task Management
- Add task with date & time
- Edit existing task
- Delete task
- Mark task as completed / undo

### 🔔 Smart Reminder System
- Uses Browser Notification API
- Automatically triggers reminder when scheduled time arrives

### 💾 Persistent Storage
- Saves tasks in localStorage
- Data remains after page refresh

### 📱 Responsive Design
- Works on mobile and desktop
- Clean modern UI
